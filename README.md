# Drowsiness detection algorithm <br>
개요: 졸음이란, 자연적인 생리 현상으로써 휴식을 필요로 할 때 나타나는 현상이다. <br> 
그러나 때론 졸음을 이기고 하던 일이나 행동을 지속해야할 때가 있다. <br>
따라서 카메라와 CO2센서를 이용해 사용자의 졸음을 감지하여 여러 상황에 맞춰 사용할 수 있다. <br>

---

## 졸음 감지 알고리즘
1. 멤버
2. 플로우 차트
3. 각 설명
